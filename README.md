## Which APK Should I Download?

| Filename Suffix | Architecture | Best For |
|-----------------|-------------|----------|
| *(no suffix)* | Universal / Pure-Java | All Android devices |
| `-arm` | 32-bit ARM | Phones before ~2016 |
| `-arm64` | 64-bit ARM ⭐ **Recommended** | Most modern Android phones |
| `-x86` | 32-bit x86 | Older emulators |
| `-x64` | 64-bit x86 | Modern emulators, Intel Chromebooks |
| `-universal` | All architectures | Any device (largest file) |

**Quick rule:** If unsure, download **`-arm64`**. It works on the vast majority of Android smartphones made since 2016.
